## 📋 文档特色

### 🎨 组件指南亮点

- **图标组件**: 支持单色和多色图标，详细的使用说明和示例
- **业务组件**: 权限控制、菜单卡片、用户卡片等业务场景组件
- **基础组件**: 页面、按钮、水印等通用基础组件

### 🛠️ 工具函数指南

- **缓存管理**: 支持过期时间的本地缓存方案
- **HTTP 请求**: 统一的接口调用和错误处理
- **权限管理**: 基于角色的权限控制
- **实时通信**: GoEasy 集成方案

### 🏗️ 项目架构

- **技术栈**: Vue 3 + Uniapp + TypeScript
- **状态管理**: Pinia 2.x
- **UI 库**: uview-plus
- **构建工具**: Vite
- **代码规范**: ESLint + Prettier
